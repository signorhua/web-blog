### http协议

#### http/0.9 

    只有一个命令 get
    没有header等描述数据的信息
    服务器发送完毕，就关闭tcp连接

#### http/1.0

    增加了很多命令 post...
    增加了status code 状态码，header等
    多字符集支持等等

#### http/1.1

    持久连接，串行发送
    pipeline
    增加了host和其他一些命令

#### http/2.0

    所有数据以二进制传输
    持久连接，同一个tcp连接里面，发送多个请求，并行发送
    头信息压缩，以及推送等提高效率的功能
